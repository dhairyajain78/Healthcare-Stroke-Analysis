\# 🏥 Healthcare Stroke Analysis



\## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a healthcare stroke prediction dataset to identify patterns, trends, and factors associated with stroke occurrence.



\## 📂 Dataset

\- Healthcare Stroke Prediction Dataset

\- Features include:

&#x20; - Age

&#x20; - Gender

&#x20; - Hypertension

&#x20; - Heart Disease

&#x20; - Marital Status

&#x20; - Work Type

&#x20; - Residence Type

&#x20; - Average Glucose Level

&#x20; - BMI

&#x20; - Smoking Status

&#x20; - Stroke (Target)



\## 📊 Analysis Performed

\- Data Cleaning

\- Missing Value Treatment

\- Outlier Detection

\- Univariate Analysis

\- Bivariate Analysis

\- Correlation Analysis

\- Data Visualization



\## 🛠️ Technologies Used

\- Python

\- Pandas

\- NumPy

\- Matplotlib

\- Seaborn

\- Jupyter Notebook

\## 📈 Key Insights

\- Age has a significant relationship with stroke occurrence.

\- Hypertension and heart disease increase stroke risk.

\- Average glucose level shows a positive association with stroke.

\- Visualizations provide deeper insights into the dataset.
